## Description

Ce projet est une application Python développée avec PyCharm permettant de gérer une base de données MySQL 
contenant des informations sur des voitures. L’application utilise un fichier de configuration JSON pour se connecter à la base de données et permet d’effectuer les opérations CRUD (Ajouter, Supprimer, Modifier et Afficher).

## Fonctionnalités

* Connexion à la base de données MySQL à l’aide d’un fichier config.json
* Ajout d’une voiture dans la base de données
* Suppression d’une voiture à partir de son identifiant
* Modification des informations d’une voiture (modèle, année, prix)
* Affichage de toutes les voitures enregistrées

## Structure du projet

* config.json : contient les informations de connexion
* voiture.py : contient la classe Voiture
* crud_db.py : contient les fonctions de gestion de la base de données
* main.py : permet de tester les fonctionnalités du programme

## Objectif

L’objectif de cet atelier est d’apprendre à connecter une application Python à une base de données MySQL, 
à manipuler des données avec les opérations CRUD et à utiliser Git et GitHub pour gérer le projet.
